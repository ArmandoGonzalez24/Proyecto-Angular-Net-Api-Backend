Se entrega el proyecto del Módulo 3 (.NET + Angular) de Talentos digitales hecho por el grupo de Armando Gonzalez, Cesar Alejandro Galarza, Fiama Fortunato y Santiago Nuñez. Para el mismo, se creó la pagina de compras de una librería, con funciones distintas para administrador (agregar, quitar o modificar productos) y usuario (agregar elementos al carrito, realizar compra).

Para acceder al proyecto se deben descargar los siguientes repositorios (mediante el comando git clone <link>) y seguir los pasos correspondientes.


https://github.com/ArmandoGonzalez24/Proyecto-Angular.Net

ng serve

Abrir http://localhost:4200

Aquí se puede interactuar directamente con la tienda y navegar sus secciones como admin o user.


https://github.com/ArmandoGonzalez24/Proyecto-Angular-Net-Api-Backend

dotnet run

Abrir http://localhost:5287/swagger/index.html

Aquí se podrá utilizar swagger para editar contenidos de la tienda como admin.
